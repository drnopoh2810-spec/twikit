📚 دليل الاستخدام

1. نشر تغريدة نصية

curl -X POST https://your-api.com/api/tweet \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "accountId": "account-id",
    "text": "مرحباً بالعالم!"
  }'

2. نشر تغريدة مع صورة

curl -X POST https://your-api.com/api/tweet/media \
  -H "X-API-Key: YOUR_API_KEY" \
  -F "accountId=account-id" \
  -F "text=تغريدة مع صورة" \
  -F "media=@image.jpg"

3. نشر تغريدة مع فيديو

curl -X POST https://your-api.com/api/tweet/media \
  -H "X-API-Key: YOUR_API_KEY" \
  -F "accountId=account-id" \
  -F "text=تغريدة مع فيديو" \
  -F "media=@video.mp4"

4. استخدام JavaScript

const apiKey = 'YOUR_API_KEY';
const accountId = 'account-id';

// نشر تغريدة نصية
const response = await fetch('/api/tweet', {
  method: 'POST',
  headers: {
    'X-API-Key': apiKey,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    accountId: accountId,
    text: 'مرحباً من JavaScript!'
  })
});

const result = await response.json();
console.log(result);

5. استخدام Python

import requests

api_key = 'YOUR_API_KEY'
account_id = 'account-id'

# نشر تغريدة نصية
response = requests.post(
    'https://your-api.com/api/tweet',
    headers={'X-API-Key': api_key},
    json={
        'accountId': account_id,
        'text': 'مرحباً من Python!'
    }
)

print(response.json())
